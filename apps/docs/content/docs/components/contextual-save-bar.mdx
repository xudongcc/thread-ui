---
title: ContextualSaveBar
description: A contextual save bar that displays save or discard options when users make changes to a form on the page.
---

<Preview path="contextual-save-bar" />

## Installation

```bash
pnpm dlx shadcn@latest add @thread-ui/contextual-save-bar
```

## Usage

```tsx
import {
  ContextualSaveBar,
  ContextualSaveBarMessage,
  ContextualSaveBarActions,
  ContextualSaveBarDiscard,
  ContextualSaveBarSave,
} from "@/components/thread-ui/contextual-save-bar";
```

```tsx
<ContextualSaveBar>
  <ContextualSaveBarMessage>Unsaved changes</ContextualSaveBarMessage>
  <ContextualSaveBarActions>
    <ContextualSaveBarDiscard>Discard</ContextualSaveBarDiscard>
    <ContextualSaveBarSave>Save</ContextualSaveBarSave>
  </ContextualSaveBarActions>
</ContextualSaveBar>
```

## Best Practices

- Only show when unsaved changes exist
- Use short, helpful messages following the "adjective + noun" pattern (e.g., "Unsaved changes")
- Save all changes on the page simultaneously; avoid multiple independent save bars on a single page
- Use strong action verbs ("Save", "Discard") instead of nouns
