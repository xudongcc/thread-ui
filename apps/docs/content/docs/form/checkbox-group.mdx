## Installation

```bash
pnpm dlx shadcn@latest add @thread-ui/checkbox-group
```

## Usage

```tsx
import {
  CheckboxGroup,
  CheckboxGroupItem,
} from "@/components/thread-ui/checkbox-group";
```

Props-first:

```tsx
<CheckboxGroup
  label="Options"
  description="Select your preferred options"
  defaultValue={["option1"]}
  parent={{ label: "All options" }}
  items={[
    { value: "option1", label: "Option 1" },
    { value: "option2", label: "Option 2" },
    { value: "option3", label: "Option 3" },
  ]}
/>
```

Compound-first:

```tsx
<CheckboxGroup allValues={["option1", "option2", "option3"]}>
  <CheckboxGroupItem parent>All options</CheckboxGroupItem>
  <CheckboxGroupItem value="option1">Option 1</CheckboxGroupItem>
  <CheckboxGroupItem value="option2">Option 2</CheckboxGroupItem>
  <CheckboxGroupItem value="option3">Option 3</CheckboxGroupItem>
</CheckboxGroup>
```

## Features

- **Collective State Management**: Manages the state of multiple checkboxes as a group
- **Props-first & Compound-first**: Use the `items` prop or compose `CheckboxGroupItem` children
- **Parent Checkbox**: Support for a parent checkbox that can select/deselect all children
- **Indeterminate State**: Automatically shows indeterminate state on parent when partially selected
- **Controlled & Uncontrolled**: Works in both controlled and uncontrolled modes
- **Built on Base UI**: Leverages Base UI primitives for accessibility

## Best Practices

- Use clear labels for each checkbox option
- Group related checkboxes together with a descriptive legend
- Use the parent checkbox pattern for "select all" functionality
- Consider using `FieldSet` and `FieldLegend` for proper form structure
