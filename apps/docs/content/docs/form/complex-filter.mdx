```bash
pnpm dlx shadcn@latest add @thread-ui/complex-filter
```

## Usage

```tsx
import {
  ComplexFilter,
  ComplexFilterLogical,
  ComplexFilterType,
  type ComplexFilterItem,
  type ComplexFilterValue,
} from "@/components/thread-ui/complex-filter";
```

```tsx
const filters: ComplexFilterItem[] = [
  {
    field: "name",
    label: "Name",
    type: ComplexFilterType.STRING,
    render: ({ value, disabled, onChange }) => (
      <Input
        disabled={disabled}
        value={value ?? ""}
        onChange={(event) => onChange(event.target.value)}
      />
    ),
  },
];

const value: ComplexFilterValue = {
  [ComplexFilterLogical.AND]: [{ name: { $eq: "Joe" } }],
};

<ComplexFilter filters={filters} value={value} onChange={setValue} />;
```

Use `NumberInput`, `DateInput`, or your own form controls inside each
field's `render` callback.

### ComplexFilterValue

`ComplexFilterValue` is a recursive group or condition object. Groups use
`$and` or `$or`; conditions use a field name with one operator/value pair.
