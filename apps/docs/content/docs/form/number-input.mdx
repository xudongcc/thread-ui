## Installation

```bash
pnpm dlx shadcn@latest add @thread-ui/number-input
```

## Usage

```tsx
import { NumberInput } from "@/components/thread-ui/number-input";
```

```tsx
<NumberInput
  thousandSeparator=","
  prefix="$"
  value={value}
  onValueChange={(values) => setValue(values.value)}
/>
```
