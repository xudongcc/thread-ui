## Installation

```bash
pnpm dlx shadcn@latest add @thread-ui/radio-group
```

## Usage

```tsx
import { RadioGroup, RadioGroupItem } from "@/components/thread-ui/radio-group";
```

Props-first:

```tsx
<RadioGroup
  label="Shipping Method"
  description="Choose how your order should be delivered."
  defaultValue="standard"
  items={[
    {
      value: "standard",
      label: "Standard",
      description: "Delivery in 5-7 business days.",
    },
    {
      value: "express",
      label: "Express",
      description: "Delivery in 2-3 business days.",
    },
    {
      value: "overnight",
      label: "Overnight",
      description: "Delivery by next business day.",
    },
  ]}
/>
```

Compound-first:

```tsx
<RadioGroup
  label="Shipping Method"
  description="Choose how your order should be delivered."
  defaultValue="standard"
>
  <RadioGroupItem description="Delivery in 5-7 business days." value="standard">
    Standard
  </RadioGroupItem>
  <RadioGroupItem description="Delivery in 2-3 business days." value="express">
    Express
  </RadioGroupItem>
  <RadioGroupItem
    description="Delivery by next business day."
    value="overnight"
  >
    Overnight
  </RadioGroupItem>
</RadioGroup>
```

## Features

- **Single Selection**: Only one option can be selected at a time
- **Props-first & Compound-first**: Use the `items` prop or compose `RadioGroupItem` children
- **Controlled & Uncontrolled**: Works in both controlled and uncontrolled modes
- **Accessible**: Proper fieldset semantics with label and description support
- **Keyboard Navigation**: Full keyboard support for option selection

## Best Practices

- Use clear labels for each radio option
- Use the `label` and `description` props to provide context
- Add descriptions to individual items when additional context is needed
- Use radio groups when users need to select exactly one option from a list
