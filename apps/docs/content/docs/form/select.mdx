## Installation

```bash
pnpm dlx shadcn@latest add @thread-ui/select
```

## Usage

```tsx
import { Select } from "@/components/thread-ui/select";
```

```tsx
<Select
  label="Country"
  description="Select your country of residence"
  placeholder="Choose a country"
  items={[
    { value: "us", label: "United States" },
    { value: "uk", label: "United Kingdom" },
    { value: "ca", label: "Canada" },
  ]}
/>
```

## Features

- **Label & Description**: Built-in support for labels and helper text
- **Error States**: Display error messages with visual feedback
- **Accessible**: Proper ARIA attributes and label associations
- **Keyboard Navigation**: Full keyboard support for option selection
- **Controlled & Uncontrolled**: Works in both controlled and uncontrolled modes

## Best Practices

- Always provide a `label` for accessibility
- Use `description` to provide helpful context about the selection
- Use `error` to display validation errors
- Provide clear and concise option labels
- Use `placeholder` to indicate what should be selected
